GoldenConnect - Bridging Generations Through Technology
=====================================================

Welcome to GoldenConnect!

GoldenConnect is a desktop application designed to connect youth volunteers 
with elderly residents in retirement homes and orphanages.

Features:
- User authentication and registration
- Session booking and management
- Activity tracking and reporting
- Elderly profile management
- Volunteer management
- Comprehensive reporting system

System Requirements:
- Windows 10 or higher
- Java 17 or higher (included in installer)
- MySQL 8.0 or higher (separate installation required)

Installation:
1. Run the installer and follow the setup wizard
2. Install MySQL database (see DATABASE_SETUP.md)
3. Configure database connection
4. Launch GoldenConnect from Start Menu or Desktop

For support and documentation, please refer to DATABASE_SETUP.md

Thank you for using GoldenConnect!
